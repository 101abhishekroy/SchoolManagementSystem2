School Management System

This project is a web-based school management system built with ASP.NET Core and Entity Framework. It provides a centralized platform to manage all school operations.

Features

Attendance tracking

Course management

Reports generation

Student and teacher management

Role-based authorization

Tech Stack

ASP.NET Core (MVC)

Entity Framework Core

SQL Server

Identity Framework for authentication

Role-based authorization

Data validation

Getting Started

Clone the repository.

Set up the SQL Server database.

Configure the connection string in the appsettings.json file.

Run the project and use the built-in authentication to log in and explore the system.
